Rendition Interface Reference
Navigator

Description

The fundamental interface for the Readium implementation.

Represents the current open EPUB rendition. Provides access to the rendition's parsed publication and its contents as well as rendering, etc. In addition, it provides access to the underlying rendering and navigation engine (typically a browser engine).
