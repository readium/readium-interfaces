◆ hasFeature()

bool ReadingSystemObject::hasFeature ( string  feature,
optional string  version 
)

General query method.

Returns true or false depending on whether the feature is supported by the current reading system.

As of 3.1, the queries that should be supported include:

dom-manipulation
layout-changes
touch-events
mouse-events
keyboard-events
spine-scripting
Parameters
featurethe feature to be tested for
versionthe version of the feature to be tested for (discouraged)
Returns
true if feature supported else false
